# 🌸 About this project

**analytic-reports-by-jooh** brings together practical examples of:

- 💻 SQL and HQL analytical queries
- 📊 BI-ready analytical data structures
- ☕ Backend development with Java 8
- 🧠 Data processing and manipulation
- 📈 Indicators, metrics, and KPIs
- 🎨 Layouts built in JasperReports Studio
- 📑 Analytical and operational reports
- ⚡ Query optimization & performance
- 🔍 Practical BI logic and report generation examples

The project acts as a mini professional framework in BI/Analytics (Java 8 + HQL + SQL + JasperReports) with reusable samples for portfolio, documentation, and learning.

> Note: All examples are anonymized. No real data, company names, internal identifiers, or PII are used.

# 💡 How to run/test the samples

1. Navigate to the desired folder (e.g., `sql/`).
2. Open and review the scripts, models, or Java code of interest.
3. To compile or test Java, use Maven (`cd java && mvn test`).
4. To visualize JasperReports templates, open `.jrxml` in Jaspersoft Studio or a compatible tool.
5. For dashboard samples, view the `.md` files in `dashboard-examples/`.

*All examples use generic data. Adapt to your environment if needed!*
