# JasperReports

Exemplos de layouts `.jrxml` genéricos para relatórios analíticos.

## Arquivos

- `relatorio_kpi_vendas.jrxml`: relatório mensal de indicadores de vendas com campos compatíveis com `KpiResult`.
- `relatorio_sla_atendimento.jrxml`: relatório operacional por fila de atendimento.

Os layouts não incluem marcas, logotipos ou dados reais.
