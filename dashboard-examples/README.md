# Dashboard examples

Ideias de painéis BI organizadas por tema. Todos os exemplos usam nomenclatura genérica e podem ser adaptados para diferentes ferramentas de visualização.

## Temas disponíveis

- `vendas-executivo.md`: visão executiva de receita, pedidos e clientes ativos.
- `operacional-sla.md`: acompanhamento de filas, produtividade e SLA.
- `retencao-clientes.md`: coortes, recorrência e segmentos.
