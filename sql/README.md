# SQL

Exemplos genéricos e anonimizados de consultas analíticas para BI.

## Convenções

- Tabelas e colunas usam nomes fictícios, como `dim_cliente`, `fato_vendas` e `dim_tempo`.
- Nenhum script contém dados pessoais reais, nomes de empresas, identificadores internos ou informações confidenciais.
- Os exemplos priorizam clareza, boas práticas de agregação e uso de CTEs para facilitar manutenção.
